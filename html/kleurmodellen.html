Kleurmodellen

Niet alleen letters en cijfers, maar ook kleuren worden gecodeerd in een computer.

Pixels

Een digitale afbeelding bestaat uit pixels. Het woord pixel bestaat uit de woorden picture en element. Als je inzoomt op een foto zie je verschillende pixels (hokjes) met verschillende kleuren.

Resolutie

Resolutie is een soort maatstaf waarmee je kan meten hoeveel pixels er op het scherm zijn, zoals 640 x 480.

RGB model

Het RGB model staat voor de kleuren rood, groen en blauw. Deze 3 kleuren vormen de basis voor alle andere kleuren. Als je nog verder zou kunnen inzoomen op je beeldscherm kun je rode, groene en blauwe lampjes zien. Elke pixel op het beeldscherm heeft deze 3 lampjes. Door de lampjes sterker of zwakker te laten branden krijg je verschillende kleuren op je beeldscherm. In het plaatje kan je bijvoorbeeld zien dat als alle 3 de kleuren even fel branden, dan krijg je de kleur wit. Je voegt kleuren toe aan een zwart achtergrond.
